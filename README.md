# COVID- 19 Thematic Maps (2020)

This project contains two interactive web maps showing county-level COVID-19 patterns in the United States (2020):

- **Map 1 (Choropleth):** COVID-19 case rates by county
-  https://lidyag.github.io/covid-map-project/map1.html

- **Map 2 (Proportional Symbols):** Total COVID-19 cases by county
-  https://lidyag.github.io/covid-map-project/map2.html
-  <img width="3811" height="1772" alt="image" src="https://github.com/user-attachments/assets/6b264df8-1f6e-44c3-99de-9335602b435c" />


- ## Tools Used to create the maps
- Live Server
- VS Code
- Mapbox GL JS
- HTML
- CSS
- Mapshaper

   **Data Source**
  - COVID-19 case data: The New York Times (2020)
  - Population data: 2018 American Community Survey (ACS) 5-Year Estimates
  - County boundary data: U.S. Census Bureau
    
  - ## Map Design
  - The map displays data using color, includes a legend for interpretation, and provides interactive click features.
